# Echos Unheard

We forget things. We forget people. There's so many people that got ignored in the history. I want to bring them back.

Used elevenlabs AI to implement the voice and prompt.

Featured Characters:
Chinese Transcontinental Railroad Worker
World War I Frontline Nurse
Immigrant at Ellis Island
Black sharecropper in the American South
Soviet Soldier in World War II
Ordinary U.S. Citizen during the Cold War

How to use: Download code zip and extract them into a folder. Open a terminal and navigate to the folder. Type "npm start" to run, then go to your local host to chat with the characters!
